# SQL Learning Adventure

> Today marks the beginning of my SQL learning journey! As someone who has mainly worked with Excel and just started exploring Python, I decided it's time to dive into SQL – the language that powers data queries in database management systems. 

---

## Why am I learning SQL?

- **Universal skill:** Almost every Data Analyst job posting requires SQL
- **Direct database access:** No more waiting for IT to export data files
- **Efficiency:** Query millions of rows much faster than Excel
- **Foundation for advanced analytics:** Many BI tools and Data Science workflows rely on SQL
- **Career growth:** SQL is a fundamental skill for Data Analysts, Data Scientists, and Business

---

## Setting up my learning environment
Instead of diving straight into complex databases, I started with a sample database that includes realistic business data from [Xóm Data](https://www.facebook.com/groups/1707916343455196). This approach helps me understand how SQL applies to real-world scenarios I might encounter as an analyst.

You guy can read more [here](https://www.facebook.com/groups/xomdata/permalink/1714685529444944) if you want to connect to our group and learn more about SQL.
